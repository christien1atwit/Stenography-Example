Use Steno.java to create an image with a hidden integer
	Steno.java will take an image in the same directory named 'sample.png' and output 'out.png'
Use StenoRead.java to read an image created with Steno.java
	StenoRead.java will take on image in the same directory named 'out.png' and then output the hidden integer into the console




---NOTE---
Steno.java - You must edit the source code to change: Input Image, Output Image, Hidden Number
StenoRead.java - You must edit the source code to change: Input Image
